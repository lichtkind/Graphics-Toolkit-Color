=head1 NAME

Graphics::Toolkit::Color::Manual::CheatSheet - GTC keywords at a glance

=head1 METHODS

   Value:      values             name / closest_name     distance
               is_in_gamut

   Color:      lighten / darken   saturate / desaturate   tint / tone / shade
               lightness          saturation              contrast      vibrance
               mix                invert                  derive        tone_curve

    Sets:      complement         triadic                 tetradic / rectangular
               analogous          cluster                 cubehelix / gradient


=head1 ARGUMENTS

     in        : (all except)     name   closest_name
     raw       : (all except)     name   closest_name   distance
     from      :                  name   closest_name
     color     : new  is_in_gamut (sub)
     range     : new  values             derive         distance
     by        : lighten    darken   saturate   desaturate   tint   tone   shade      mix
                 lightness           saturation              contrast      vibrance   derive
     multiply  : lightness           saturation              contrast      vibrance   derive
     set       : lightness           saturation              contrast      vibrance   derive
     only      :                  distance   invert
     to        :                  distance   mix          analogous               gradient
     steps     :                             complement   analogous   cubehelix   gradient 
     tilt      :                             complement   analogous   cubehelix   gradient 


=head1 FORMATS

              list  (1,2,3)
             array  [1,2,3]
       named_array  [RGB =>  1,2,3 ]
      nested_array  [RGB => [1,2,3]]
              hash  {red => 1, green => 2, blue => 3}
         char_hash  {r   => 1, g     => 2, b    => 3}
      named_string  'rgb: 1, 2, 3'  
        css_string  'rgb( 1, 2, 3)' 
        hex_string  '#010203' '#123' (RGB only)


=head1 SPACES

      RGB : RGB / LinearRGB / CIEXYZ / CIERGB 
            AdobeRGB / AppleRGB / ProPhotoRGB / WideGamutRGB 
            Display P3 / Display P3 Linear / DCI-P3 / Display P3 Linear / Rec.709 / Rec.2020
      LAB : OKLAB / CIELAB / HunterLAB / CIELUV / YIQ / YCbCr / YPbPr
      HSL : OKHSL / OKLCH / HSL / CIELCHab / CIELCHuv
      HSV : OKHSV / HSV / HSB
      HWB : OKHWB / HWB / NCol
    other : CMY / CMYK / Yxy
